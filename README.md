Fitting Emission Lines from SOFIA FIFI-LS Data

FIFI-LS_Line_Fitting.ipynb is broken up into the following parts:
1. Accessing Archival Data from the NASA/IPAC Science Archive
2. Using the SOFIA SPectral EXplorer
3. Environment Setup and Data Initialization
4. Continuum Fitting & Subtraction
5. Gaussian Fitting & Exporting Results


Dependencies: <br />
astropy 7.2.0: Core framework for astronomical coordinates <br />
specutils 2.3.0: Toolkit for spectroscopic data analysis and fitting <br />
astroquery 0.4.11: Interface for programmatic access to the NASA/IPAC NED database <br />
scipy 1.17.0: Numerical optimization engine used for the Gaussian parameter estimation <br />
matplotlib 3.10.8: Engine for generating publication-quality plots <br />
numpy 2.4.2: Mathematical functions <br />
pandas 3.0.1: Data manipulation and analysis <br />

This notebook is optimized for Python v3.11.14
